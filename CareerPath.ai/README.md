# CareerPath.ai

**Find your perfect career — 100% offline, AI-style, no APIs.**

An intelligent career advisor that matches your **skills, IQ, budget, country, and interests** to **250+ real careers** using **local logic only**.

**Includes resume builder**, **PDF export**, **dark mode**, **mobile support**, and **instant redeploy**.

---

## Live Demo
[https://careerpath-ai.vercel.app](https://careerpath-ai.vercel.app)

---

## Features

- AI-style career matching (client-side)
- 250+ career dataset with salary, roadmap, growth
- Resume builder with auto-fill & PDF export
- Dark / Light mode toggle
- Save results in `localStorage`
- Share on X / LinkedIn
- Fully static — deploy on **Vercel**, **GitHub Pages**, **Netlify**

---

## Tech Stack

- **Next.js 14** (App Router)
- **React 18**
- **Pure CSS** (no Tailwind)
- **jsPDF** (PDF export)
- **localStorage** (save & resume)
- **No APIs, no backend**
